## [0.1.0] - 2025-02-28

### 新增
- 初始版本发布
- 可视化图形界面
- 鼠标和键盘操作录制
- 动作组管理功能
- 脚本执行和循环控制
- 执行速度调节（0.1x - 10x）
- 状态自动保存和恢复
- 窗口绑定功能
- 脚本导出为 Python
- 示例脚本（三国杀、神风录等）
- 完整的用户文档

### 支持的动作类型
- 鼠标点击（左键、右键、双击）
- 键盘输入
- 图像识别点击
- 延时等待
- 文本复制/粘贴
- 窗口操作

### 技术特性
- 基于 PyQt5 的图形界面
- 使用 pyautogui 进行自动化操作
- 使用 pynput 进行输入设备控制
- 支持 Pixi 包管理器
- 可打包为独立可执行文件

### 文档
- README.md - 项目介绍和快速开始
- USER_GUIDE.md - 详细用户指南
- CONTRIBUTING.md - 贡献指南
- CHANGELOG.md - 变更日志

## 版本说明

### 版本号格式
- 主版本号：不兼容的 API 修改
- 次版本号：向下兼容的功能性新增
- 修订号：向下兼容的问题修正

### 变更类型
- **新增** - 新功能
- **变更** - 功能变更
- **弃用** - 即将移除的功能
- **移除** - 已移除的功能
- **修复** - Bug 修复
- **安全** - 安全相关修复
